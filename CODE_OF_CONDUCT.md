# Code of Conduct

This project follows a professional, respectful collaboration standard.

## Expected Behavior

- Be respectful and constructive in issues, PRs, and reviews.
- Focus criticism on code and decisions, not individuals.
- Assume good intent and ask clarifying questions before escalating.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Doxxing, intimidation, or threats
- Repeated disruptive behavior after warning

## Enforcement

Maintainers may take any action they consider appropriate, including warning,
temporary moderation, or removal from project spaces.

## Reporting

Report conduct concerns privately to maintainers through the same private path
used for sensitive repository concerns.
