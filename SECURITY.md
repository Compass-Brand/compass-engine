# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository or distributed tooling:

1. Do not open a public issue with exploit details.
2. Report privately to the Compass Brand maintainers.
3. Include:
   - affected files/components
   - reproduction steps
   - impact assessment
   - suggested mitigation (if available)

## Response Targets

- Initial acknowledgment: within 3 business days
- Triage decision: within 7 business days
- Fix/mitigation timeline: based on severity and release impact

## Scope

Security reports may include:
- secret exposure risks
- workflow/action supply chain risks
- unsafe default configuration in distributed bundles
- privilege escalation paths in automation scripts
