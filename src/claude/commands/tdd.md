# TDD

## Usage

/tdd <file-or-feature>

## What This Command Does

1. Invokes the tdd-guide agent
2. Applies tdd-workflow skill methodology
3. Enforces RED-GREEN-REFACTOR cycle
4. Ensures test is written before implementation

## Invoked Components

- Agent: tdd-guide
- Skill: tdd-workflow

## Example

```
/tdd src/services/auth.ts
```
