## Section Rationale

- End-to-end verification: confirms complete success.
- Next steps: supports progressive disclosure into deeper docs.
- Outcomes: defines measurable learning target.
- Prerequisites: avoids setup failure before core learning starts.
- Step checkpoints: keeps readers aligned and reduces silent failure.
- Troubleshooting: addresses predictable blockers without external escalation.
- What you built: reinforces mental model and retention.
