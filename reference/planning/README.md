# Planning Reference Framework

This directory is the planning reference index for Compass Engine.

## Purpose

- Provide a stable entrypoint for planning structure guidance.
- Separate top-level navigation from the actual planning model.
- Keep planning practices consistent for brownfield BMAD continuation.

## Domain

`framework/` is the canonical planning framework domain.
`docs/` contains guidance and rationale for using that framework.

Read order:
1. `framework/README.md`
2. `docs/how-to-use.md`
3. `framework/current/phase.md`

reference/planning/
